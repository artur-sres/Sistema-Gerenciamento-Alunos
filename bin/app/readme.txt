- Devo ser capaz de criar uma turma que vai possuir um array de alunos 
- Devo ser capaz de visualizar relatorios sobre a turma como: aluno(os) com maior média, média da turma, quantidade em cada situação, entre outros

- Devo ser capaz de cadastrar um aluno em uma turma
- Cada aluno deve possuir:
    nome   
    matricula
    dataNascimento
    idade
    média
    período
    sexo(M/F)
    situação(Aprovado, Final, Reprovado)
    
- Devo ser capaz de buscar um aluno em especifico
- Devo ser capaz de exibir uma lista com todos os alunos
- Devo ser capaz de excluir algum aluno
- Devo ser capaz de atualizar os dados de um aluno
- Devo ser capaz de listar os alunos por situação

O sistema deve possuir um menu que possibilite o usuario de conseguir interagir

Implemente um código em Java para realizar o controle de alunos de uma escola. Cada aluno deve possuir o nome, matricula, idade, média, período, sexo(M/F), situação (aprovado, final e reprovado), entre outros. Adicionalmente, é preciso realizar o gerenciamento dos alunos, podendo: adicionar, buscar, exibir todos os alunos, excluir e atualizar um aluno, listar por situação e exibir relatório (aluno(os) com maior média, média da turma, quantidade em cada situação, entre outros). Criar um menu possibilitando que o usuário consiga interagir com o sistema.
